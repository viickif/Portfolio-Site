Sophos

Tools: C/C++, Python, Lua, Git, JIRA, Agile, Jenkins, Coverity, PyTest, Qt


At Sophos, I spent 4 months working on their new Intrusion Prevention System (IPS). It's an advanced firewall feature that offers network security to its users.

I primarily coded in C++ to develop features for the IPS. On top of that, I used C to customize Snort, the network intrusion detection system that we were using, to implement special detection types that don't disturb network traffic when triggered. I also used Qt to make helpful tools that simulated future in dev features which helped my coworkers start working on depended code earlier and streamline the development process. I even tried my hand at Lua to establish communication between different components of the umbrella system.

On a different note, I got to practice Agile in a professional setting where I enhance my abilities to articulate technical challenges and design decisions during stand ups and feature kickoffs. It was so exciting being able to see the development of this software all the way up to its last Sprint.

Not only was I introduced to the world of cybersecurity and networking, this experience really improved my abilities to optimize the performance of my programs since IPS operations were often resource intensive. The wonderful Senior Engineers over at Sophos always provided a lot of insightful comments during my code reviews that taught me so much about object oriented programming best practices.

Clash of Snakes

Tools: JavaScript, Node.js, Express.js, Socket.IO, HTML/CSS


Clash of Snakes is a 2 player online game based on the classic snake game. Socket.IO, Node.js and Express.js were used to establish bi-directional connection between web clients and the server.

A group of friends and I thought that it would be fun to make an interactive game after reminiscing about Wii Sports from our childhood. To do this, we made a hand gesture controlled remote that connected to a Raspberry Pi and I implemented some Python code that would handle the sensor control logic and communicate with our web server.

It was really challenging to get the controls to sync up just right with what was happening on the screen to minimize any lag and it was all the more rewarding when I was able to find the source of the bug. I probably got some strange looks, waving my hands in the air trying to debug this project, but that's what made it so memorable!

Later, my friend and I decided to implement a keyboard controlled version of the game so that more people could access it. After I fixed the food spawn and movement synchronization issues, we even added a custom map. Users can enter their player name and select a difficulty level that determines the snake speed and points target.


For more details, checkout the keyboard controlled Clash of Snakes GitHub repo.

Algorithmic Art Image Editor

Tools: C++


Algorithmic Art Image Editor (AAIE) is a fun image editor program that I made with a partner over the course of 4 months. It taught me a lot about how to make good use of data structures and algorithms. Here's an overview of its 3 main features and some examples of its results.

Flood fill GIF animations:

Users can choose a closed region of an image to generate a GIF of flood filling it with a custom pattern. To do this, we used BFS, DFS and deques as well as inheritance/virtual functions.

Reordering image chains:

Users can apply filters and watermarks to images or choose to provide an image that we can chop up and reorder in visually interesting ways (weaving, rolling back, reversing, etc). We accomplished this by splitting the image up into strips, putting them in a circular doubly-linked list and changing pixel properties.

Blocky effect:

Users can also recreate any image using this cool blocky effect. We used a few image compression, space partition and statistical algorithm optimization concepts for this. It involved recursively splitting each image into quarters and building a quad tree, calculating color entropy, pruning the tree and more.
